# Data Folder

This folder should contain the **BIP Challenge 2017** event log file: 
``data/BPI Challenge 2017.xes``

You can download the original event log from: https://data.4tu.nl/articles/dataset/BPI_Challenge_2017/12696884

⚠️ Note: The dataset (~552 MB) is too large for GitHub and is therefore excluded via .gitignore.
